# sign-hub

个人签到工具集合。每个签到目标一个子目录, 互不干扰。

## 目录结构

```
sign-hub/
├── README.md  # 项目说明
├── .gitignore  # 忽略敏感文件
└── script_name/  # 每个签到目标独立子目录
```

## 脚本列表

| 目录 | 目标 | 状态 |
|------|------|------|
| `citybox/` | CITYBOX 魔盒微信小程序 | ✅ 已验证 |

## 通用约定

- 每个签到目标独立子目录, 自带 `README.md` 和 `config.example.json`
- 实际配置文件 (含 token 等敏感信息) 统一命名 `config.json`, 已在根 `.gitignore` 忽略
- 接口间随机延时, 避免风控

各脚本的安装、使用、抓包教程见对应子目录下的 `README.md`。
